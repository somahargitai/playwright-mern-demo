Todo API Server

Welcome to the Todo API server. This server provides endpoints to manage your todo items.

Available Endpoints:

GET /api/todos

Retrieves all todo items.

GET /api/todos/:id

Retrieves a specific todo item by ID.

POST /api/todos

Creates a new todo item. The request body should include a text property.

PUT /api/todos/:id

Updates a specific todo item. The request body can include text and/or completed properties.

DELETE /api/todos/:id

Deletes a specific todo item.

Usage Example

To create a new todo:


fetch('/api/todos', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        text: 'Learn Playwright testing'
    })
})
.then(response => response.json())
.then(data => console.log(data));

The Todo App frontend is available at: http://localhost:5173
